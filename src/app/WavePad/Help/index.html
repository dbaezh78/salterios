Technical Support

If you have difficulties using WavePad Sound Editor please read the applicable topic before requesting support. If your problem is not covered in this user guide please view the up-to-date WavePad Sound Editor Online Technical Support at www.nch.com.au/wavepad/support.html. If that does not solve your problem, you can contact us using the technical support contacts listed on that page.

If you have any suggestions for improvements to WavePad Sound Editor, or suggestions for other related software that you might need, please post it on our Suggestions page at www.nch.com.au/suggestions/index.html.
