Upgrading to WavePad Master's Edition

A number of professional tools for WavePad are only available if you have purchased an upgrade to WavePad Master's Edition. These features include the batch file processor, keyboard macros, projects, bookmarks, regions and the region assembler.

To view pricing and to purchase an upgrade to WavePad Master's Edition please open www.nch.com.au/wavepad/register.

Once you have purchased and activated your licence you will receive a registration with your name and contact details. Use the menu File -> Register Master's Edition Upgrade and enter the details exactly as they appear in the registration.
