Main Screen Reference

Wave form display

There are two wave windows in the main screen. The upper window displays the wave form of the audio file always in its full length - it is useful for knowing where you are in the file. The lower window shows the wave form of the audio based on the zoom level.

Zoom Buttons

~Zoom In:

Click on this button to zoom in.

~Zoom Out:

Click on this button to zoom out.

~Zoom To Selection:

Click on this button to zoom to the region you have just selected.

~Show Entire Region:

Click on this button to zoom out to display the entire file length.

~Vertical Zoom:

Click on this button to increase the amplitude of the display (to see softer sounds).

~Wave display + up / - down:

This combines the left and right channels into one waveform.

~Wave display left channel up / right channel down:

Displays the left channel on the upper side of the center line and the right channel on the lower side of the center line.

~Wave display dual left and right channels:

Combines the left and right channels into one waveform.

~Sample Edit Mode

Allows you to modify the individual audio samples in the waveform using the computer's mouse.

There is also a zoom track bar slider control between the Zoom Out and Zoom In buttons that provides infinitely variable zoom.

Selection Display

~Start:

This displays the start position of the selected region.

~End:

This displays the end position of the selected region.

~Sel Length:

This displays the length of the selected region.

~File Length:

This displays the length of the entire file.

To select a region, you can either click on the wave window and drag to the position you want or hold down the Shift key and press the left or right arrow buttons.
