Record Control Window

The Record Control window is where you make your recordings and listen to them. You can also specify what sound recording device you want to use as well as auto start and auto trim settings.

File Info

Name - the name of the file to which you are recording.

Playback

Device - select the playback sound device you want to use.

Volume - set the playback volume level.

Recording

Device - select the recording sound device you want to use.

Input - select the recording channel you want to use.

Volume - set the recording volume level.

Advanced Record Options:

Auto Start Recording - if selected, the recording will only start if audio is detected (i.e. it will not record if there is just silence).

Auto Trim Recording - if selected, the start and end of the recording will be trimmed of silence.

Voice Activated Recording - if selected, this will only record if the audio volume level is above what you specify for the "Threshold" setting.

The recording window also has a wave window where you will be able to see the audio waveform after it has been recorded. After recording, you can manually navigate to any portion of the audio using the scrollbar, and either play your audio or record again. Note that performing either action will either play or record the audio from the position of the green cursor in the wave window.

Along the bottom of the window you will see a number of buttons that are used for playback and recording of audio. These buttons are as follows:

Record - commences the recording. Recording will start from wherever the green cursor is located (F5).

Play - plays the recording. Playback will commence from wherever the green cursor is located (space).

Pause - Pauses playback.

Stop - stops recording or playback of the audio (Esc).

Go to Start - sets the green cursor back to the start of the audio recording (Home).

Rewind - moves backwards through the audio while playing it back (Left Arrow).

Fast Forward - moves forwards through the audio while playing it back (Right Arrow).

Go to End - sets the green cursor to the end of the audio recording (End).
