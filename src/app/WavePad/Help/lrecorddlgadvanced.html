Automatic Recording

The recording can be automatically started when either the Record Control dialog is opened or when the audio to be recorded reaches the threshold level. Additionally any quiet periods at the start and the end of the recording that is below the threshold level can be trimmed.

Auto Start Recording

The recording will automatically start when the Record Control dialog is opened, you will not need to select the record button.

Auto Trim Recording

Trim the recording to remove any quiet periods from the start and ending of the recording. Use the threshold slider to specify the cutoff level.

Voice Activated Recording

Automatically start the recording when the audio level reaches the threshold.

Threshold:

Use the slider to set the threshold level for the recording. Used for both the auto trim and the voice activated start.
