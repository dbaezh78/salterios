WavePad Options ~ Keys Tab

If you find you need to perform a number of edit or effect functions in sequence often, you can use the Keys tab of Settings to assign key macros. When you press the specified key WavePad will perform the list of functions.

Note: this feature is only available on WavePad Master's Edition. To purchase an upgrade see www.nch.com.au/wavepad/register.

To create a new key macro...

~ Open WavePad Options -> Keys.

~ Click on Add.

~ Press the key you want to assign as a macro.

~ Click Add to add a command to the list.

~ Select the command(s).

~ Click OK.

From then on, whenever you push the assigned key the list of functions will be performed on the current file.

If you have selected a function that requires data or settings you will be prompted for that data when you press the key.
